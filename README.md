
# Trust Vault

Este proyecto presenta una aplicación diseñada para ayudar a los usuarios a gestionar de forma segura sus contraseñas y analizar su fortaleza frente a posibles ataques.

La aplicación ofrece las siguientes funcionalidades principales:

- Almacenamiento seguro: Utiliza algoritmos de encriptación de última generación para proteger las contraseñas del usuario.
- Generación de contraseñas fuertes: Crea contraseñas aleatorias y complejas que cumplen con los estándares de seguridad más exigentes.
- Análisis de contraseñas: Evalúa la fortaleza de las contraseñas existentes y proporciona recomendaciones para mejorarlas.
- Detección de contraseñas comprometidas: Compara las contraseñas almacenadas con bases de datos de contraseñas filtradas para identificar posibles vulnerabilidades.
- Multiplataforma: Disponible en diversas plataformas (web, móvil, escritorio) para facilitar el acceso desde cualquier dispositivo.
- Sincronización: Permite sincronizar las contraseñas en múltiples dispositivos de forma segura.

## Features

- Modo claro/oscuro.
- Log-in y registro de usuarios.
- Control de salud de contraseñas.
- Encriptación de las claves introducidas.
- Verificación en dos pasos con código SMS.
